a. Git es un proyecto de código abierto maduro y con un mantenimiento activo que desarrolló originalmente Linus Torvalds, el famoso creador del kernel del sistema operativo Linux, en 2005. 
Git se ha ideado para posibilitar la ramificación y el etiquetado como procesos de primera importancia (a diferencia de SVN) y las operaciones que afectan a las ramas y las etiquetas (como la fusión o la reversión) también se almacenan en el historial de cambios.


b. El 4 de junio de 2018, Microsoft compró GitHub por la cantidad de 7500 millones de dólares.
Github fue comprado por microsoft porque le brindaba un acercamiento a la comunidad más vibrante en el desarrollo de software libre y abierto


c. Existe tambien la interfaz visual de github llamada Github Desktop 
Se creó como una forma de facilitar el manejo de git para gente que no domina el uso de una terminal.
