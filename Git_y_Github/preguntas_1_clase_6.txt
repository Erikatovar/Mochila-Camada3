¿Quién inventó el sistema de control de versión Git y por qué?
Linus Torvalds invento el sistema de control de version de git en 2005 para reemplazar al sistema de control de versiones que mas se usaba en el momento, bitkeeper, porque este se hizo de pago. Teniendo en mente las cualidades de bitkeeper, git se hizo con el objetivo de tener;  Velocidad, Diseño sencillo, soporte para desarollo no lineal, distribucion completa, y capacidad de manejar grandes proyectos eficientemente.
