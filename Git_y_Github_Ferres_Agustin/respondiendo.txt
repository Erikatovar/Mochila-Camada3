//---------¿Quién inventó el sistema de control de versión Git y por qué?---------//
    Linus Torvalds, el famoso creador del kernel del sistema operativo Linux, lo creo 
    para posibilitar la ramificación y el etiquetado como procesos de primera importancia
    (a diferencia de SVN) y las operaciones que afectan a las ramas y las etiquetas (como la fusión o la reversión) 
    también se almacenan en el historial de cambios
